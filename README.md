## ✨ الميزات التفصيلية

### 💰 أسعار الذهب اللحظية
- عيارات 24 / 21 / 18 — سبائك، مشغولات، جنيه ذهب
- تحديث تلقائي كل **60 ثانية** عبر API مجاني
- شريط Ticker متحرك في أعلى الصفحة
- بطاقات سعرية مع نسبة التغير ↑↓

### 📈 الرسوم البيانية التاريخية
- رسم بياني خطي / منطقة (Line / Area Chart)
- اختيار الفترة: آخر 7 / 14 / 30 يوم
- مقارنة بين عيارات متعددة في نفس الوقت
- رسم بياني للعملات الأجنبية

### 🌡️ خريطة حرارة السوق (Heatmap)
- شبكة خلايا ملونة لكل الأعيرة والعملات
  - 🟢 أخضر = ارتفاع قوي
  - 🔴 أحمر = هبوط
  - 🟡 أصفر = استقرار
- رسم بياني أفقي للأداء النسبي

### 🧮 الحاسبة الذكية
- حساب سعر الشراء مع المصنعية (10% / 15% / 20% / 25%)
- تفصيل: سعر المعدن + المصنعية + الإجمالي
- تحويل فوري: جرام ↔ أوقية ↔ قيمة بالجنيه

### 💼 محفظة المستثمر
- تسجيل مشتريات الذهب (وزن + سعر + عيار + تاريخ)
- حساب الربح / الخسارة لحظياً بالجنيه والنسبة %
- عرض الأداء الإجمالي في 4 بطاقات
- تصدير المحفظة بصيغة **Excel (CSV)**
- حفظ تلقائي في localStorage

### 🔔 نظام التنبيهات الذكية
- تنبيه عند وصول **سعر محدد** أو **نسبة تغير %**
- اتجاه: صعود ↑ أو هبوط ↓
- **تنبيه صوتي** اختياري (Web Audio API — بدون ملفات)
- إشعار داخل الموقع عند تحقق الشرط

### ⭐ قائمة المفضلة (Watchlist)
- إضافة أي عيار ذهب أو عملة للمتابعة السريعة
- بطاقات سعرية لحظية للعناصر المفضلة
- حذف فردي وحفظ تلقائي

### 🧠 التحليل الذكي المبسط
- إشارة واضحة: **وقت مناسب للشراء / انتظر / ذروة ارتفاع**
- مؤشران شريطيان: مؤشر الشراء + مؤشر الذروة (Overbought)
- 4 بطاقات بيانات فورية (تغير اليومي، فجوة الدولار، نصيحة اليوم)
- **AI Chat**: اسأل المحلل الذكي بناءً على الأسعار الحية

### ⏰ مؤشر توقيت الشراء (Smart Timing)
- بطاقة بطولية بإشارة واضحة: شراء / انتظار / بيع
- 4 عوامل مرئية: اتجاه 10 أيام، تقلب السوق، ضغط الدولار، تغير عيار 21
- تنبيهات نصية تلقائية: "السوق نشط الآن في الشراء" / "انخفاض مفاجئ"
- رسم بياني لحركة عيار 21 آخر 10 أيام

### 🔥 نبض السوق المصري (Egypt Market Pulse)
- **مؤشر دائري متحرك** (0-100) لمستوى نشاط السوق
- 3 أشرطة: ضغط شراء / ضغط بيع / استقرار
- تصنيف السوق: 🔥 نشاط عالٍ / 😐 استقرار / 📉 ركود نسبي
- 4 إشارات مباشرة بالأسعار الحية

### 📅 مواسم الذهب في مصر
- تقويم سنوي بـ **8 مواسم** مصرية:
  - رمضان والعيد 🌙، موسم الأفراح والزواج 💒
  - الصيف وموسم السفر 🌞، الشتاء والأعياد 🎄
  - عيد الحب 💕، وما بعد الأعياد، والفترات الهادئة
- تأثير كل موسم على الأسعار: 🔴 عالٍ / 🟡 متوسط / 🟢 منخفض
- بطاقة "الموسم الحالي الآن" مع وصف وتوصية
- رسم بياني للطلب الموسمي على مدار السنة
